什么是 WorkMolde AI？
WorkMolde AI 是一款基于 AI 的文档处理代理工具，能够自动读取、分析、生成和转换各类文档格式（.docx、.xlsx、.pptx、.pdf、Markdown 等）。它通过与 LLM 集成，理解用户意图并智能化地执行文档处理任务。
如何添加模型提供商？
在左侧菜单中打开"设置"，进入"LLM 配置"选项卡。点击"添加提供商"按钮，填写提供商名称、API 地址和密钥等信息，保存后点击"测试连接"验证可用性。
支持哪些文件格式？
目前支持以下文件格式：Word 文档（.docx）、Excel 工作簿（.xlsx）、PowerPoint 演示文稿（.pptx）、PDF 文件（.pdf）以及 Markdown 文件（.md）。更多格式支持正在开发中。
如何使用模板？
在输入框上方有快捷模板卡片，点击即可填充预设提示词。你也可以在"设置 → 提示词模板"中管理所有模板，支持编辑和自定义。
如何查看日志？
日志文件位于应用数据目录下的 logs/ 文件夹。在 Windows 上通常位于 %APPDATA%/WorkMolde/logs/，macOS 上位于 ~/Library/Application Support/WorkMolde/logs/。
